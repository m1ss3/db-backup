
## Database Backup

Clone the project

```bash
  git clone https://github.com/m1ss3/db-backup
```

Go to the project directory

```bash
  cd db-backup
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  yarn start
```

If you are having problems or seeing errors, definitely contact us and let's try to solve it!
